# Документация для разработчиков

**Конструктор схем ВОЛП** — вспомогательный сервис для ввода общих технических данных объекта и примитивный графический редактор для наглядного представления данных. Репозиторий в разработке.

## Разделы

- **[Архитектура](architecture.md)** — структура проекта, назначение пакетов, разделение UI и логики, потоки данных.
- **[Настройка окружения](setup.md)** — Python 3.8, venv, зависимости, запуск из исходников.
- **[Конфиги](configs.md)** — описание файлов в `configs/`, ключи и использование в коде.
- **[Сборка](build.md)** — сборка exe (Nuitka), установщик (Inno Setup), режимы demo/full.
- **[API](api/entry.md)** — автогенерируемое описание модулей и классов по docstrings (точка входа, `package.app`, `package.components`, `package.controllers`, `package.modules`, `package.ui`, `package.constants`).

## Сборка документации

Установите зависимости для документации и запустите предпросмотр или сборку:

```bash
pip install -r requirements-docs.txt
mkdocs serve
```

Откройте в браузере адрес, указанный в выводе (обычно `http://127.0.0.1:8000`). Статическая сборка в каталог `site/`:

```bash
mkdocs build
```
