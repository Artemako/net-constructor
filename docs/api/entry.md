# Точка входа

Запуск приложения выполняется из файла **main.pyw** в корне репозитория.

## Назначение

- Разбор аргументов командной строки: `--demo`, `--full` и путь к файлу `.nce` для открытия при старте.
- Установка переменной окружения `NET_CONSTRUCTOR_MODE` (`demo` или `full`) до импорта приложения.
- Вызов `package.app.App` с путём к каталогу приложения и опциональным путём к файлу проекта.

## Запуск

- Демо: `python main.pyw --demo`
- Полная версия: `python main.pyw --full`
- С открытием файла: `python main.pyw путь/к/файлу.nce`

Если флаги не указаны, режим берётся из переменной окружения `NET_CONSTRUCTOR_MODE` или из сгенерированного при сборке `package.modules._build_config`.

Далее инициализация приложения выполняется в [package.app](app.md).
